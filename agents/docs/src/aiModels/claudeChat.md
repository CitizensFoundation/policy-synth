# ClaudeChat

The `ClaudeChat` class provides an interface for interacting with Anthropic's Claude models using a chat-based paradigm. It extends the `BaseChatModel` and supports both streaming and non-streaming completions, tool/function calling, token usage tracking, and system message handling. This class is designed to be used as a backend model adapter for AI chat agents.

---

## Properties

| Name               | Type                | Description                                                                                 |
|--------------------|---------------------|---------------------------------------------------------------------------------------------|
| client             | Anthropic           | The Anthropic SDK client instance used to communicate with Claude models.                   |
| maxThinkingTokens  | number \| undefined | Optional maximum number of "thinking" tokens for Claude's internal reasoning.               |
| config             | PsAiModelConfig     | The configuration object for the Claude model, including API key, model name, etc.          |
| modelName (inherited) | string           | The name of the Claude model being used (e.g., "claude-3-opus-20240229").                   |
| maxTokensOut (inherited) | number        | The maximum number of output tokens for a completion.                                       |
| logger (inherited) | any                 | Logger instance for debug and info output.                                                  |
