# OpenAiChat

A class for interacting with OpenAI's chat completion API, supporting both streaming and non-streaming responses, tool calling, and advanced configuration for reasoning and logit bias. This class extends `BaseChatModel` and is designed to be used as a model adapter within the PolicySynth agent framework.

**File:** `@policysynth/agents/aiModels/openAiChat.js`

---

## Properties

| Name   | Type                | Description                                                                                 |
|--------|---------------------|---------------------------------------------------------------------------------------------|
| client | `OpenAI`            | The OpenAI SDK client instance used to make API calls.                                      |
| cfg    | `PsOpenAiModelConfig` | The configuration object for the OpenAI model, including API key, model name, and options. |

---
