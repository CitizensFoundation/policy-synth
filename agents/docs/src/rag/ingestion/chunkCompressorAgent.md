# IngestionChunkCompressorAgent

The `IngestionChunkCompressorAgent` is a specialized agent for compressing large text chunks while preserving their meaning, nuance, and detail. It uses an LLM (Large Language Model) to compress each paragraph of the input text and validates the compressed output for completeness, correctness, and absence of hallucinated content. The agent retries compression with improved instructions if validation fails, up to a configurable maximum number of attempts.

This class extends `BaseIngestionAgent`.

---

## Properties

| Name                                      | Type      | Description                                                                                      |
|-------------------------------------------|-----------|--------------------------------------------------------------------------------------------------|
| maxCompressionRetries                     | number    | Maximum number of compression attempts before giving up. Default: 15.                            |
| retryCountBeforeRandomizingLlmTemperature | number    | Retry count after which LLM temperature may be randomized. Default: 13.                          |
| completionValidationSuccessMessage         | string    | Success message for completion validation.                                                       |
| correctnessValidationSuccessMessage        | string    | Success message for correctness validation.                                                      |
| hallucinationValidationSuccessMessage      | string    | Success message for hallucination validation.                                                    |
| hallucinationValidationSystemMessage       | string    | System prompt for hallucination validation.                                                      |
| correctnessValidationSystemMessage         | string    | System prompt for correctness validation.                                                        |
| completionValidationSystemMessage          | string    | System prompt for completion validation.                                                         |
| validationUserMessage                      | function  | Function to generate user message for validation.                                                |
| compressionSystemMessage                   | string    | System prompt for initial compression.                                                           |
| compressionRetrySystemMessage              | string    | System prompt for retrying compression with suggestions.                                         |
| compressionUserMessage                     | function  | Function to generate user message for initial compression.                                       |
| compressionRetryUserMessage                | function  | Function to generate user message for retrying compression with suggestions and previous results. |
