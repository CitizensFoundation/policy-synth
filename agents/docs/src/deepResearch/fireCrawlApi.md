# FirecrawlScrapeAgent

The `FirecrawlScrapeAgent` is a specialized agent for scraping web pages using the [Firecrawl API](https://firecrawl.dev/) within the PolicySynth agent framework. It can extract content in various formats, filter out privacy policy or terms of service documents, and handle rate-limiting gracefully. It extends the `PolicySynthAgent` base class.

## Properties

| Name             | Type                                 | Description                                                                                 |
|------------------|--------------------------------------|---------------------------------------------------------------------------------------------|
| needsAiModel     | `boolean`                            | Indicates if the agent requires an AI model (always `false` for this agent).                |
| app              | `FirecrawlApp`                       | Instance of the Firecrawl client used for scraping.                                         |
| crawlPageLimit   | `number`                             | Maximum number of pages to crawl in a single operation (default: 50).                       |

## Constructor

```typescript
constructor(
  agent: PsAgent,
  memory: PsAgentMemoryData | undefined,
  startProgress: number,
  endProgress: number
)
```

- **agent**: The agent database model instance.
- **memory**: Optional agent memory data.
- **startProgress**: Progress value at the start of the operation.
- **endProgress**: Progress value at the end of the operation.

**Throws**: Error if the `FIRECRAWL_API_KEY` environment variable is not set.
